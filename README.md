#Uasge
カテゴリ別に食事を閲覧することが出来ます。
カテゴリは以下の５つ
* Morning
* Afternoon
* Evening
* Recently(90日以内に投稿されたもの)
* Top rate(評価3.5以上)


また、トップページのフォームから新たに食事を追加したり
評価をすることが出来ます。
